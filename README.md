# LiraMed - Distribuidora Hospitalar

A LiraMed é uma distribuidora hospitalar dedicada a fornecer produtos confiáveis e de alta qualidade para o setor da saúde. Este projeto é um website institucional desenvolvido com HTML, CSS e JavaScript, focado na experiência do usuário e em recursos modernos como flipbook interativo e design responsivo.

## 🚀 Funcionalidades

- **Design Moderno**: Interface limpa e profissional com modo escuro automático.
- **Flipbook Interativo**: Navegação intuitiva no catálogo com efeito de virar páginas e zoom.
- **Design Responsivo**: O site se adapta perfeitamente a diferentes dispositivos (desktop, tablet, mobile).
- **Formulário de Contato**: Envie mensagens diretamente pelo site.
- **Botões de Ação Rápida**: Links diretos para WhatsApp, Instagram e E-mail.
- **Animações CSS**: Transições suaves e efeitos visuais para uma melhor experiência.

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

O site foi desenvolvido com foco em mobile-first, garantindo uma excelente experiência em qualquer tamanho de tela:

- **Mobile**: Layout de coluna única com navegação compacta.

- **Desktop**: Layout expandido com todos os recursos visuais.
